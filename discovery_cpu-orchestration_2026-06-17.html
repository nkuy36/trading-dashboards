Thesis

Agentic AI workloads consume 15x more tokens per user and run continuously vs. chatbot bursts. CPU represents ~90% of latency in agent loops; per Arm, CPU cores per GW are going from 30M to 120M (4x). The CPU:GPU ratio is shifting from 1:8 toward 1:1, and this demand inflection collides with TSMC advanced-node capacity routed to AI accelerators.

Time horizon: multi-year structural. Classification: EARLY.

⚠ Thesis-shifting development since last update (21 Apr)

NVIDIA put Vera — a CPU built explicitly to orchestrate agents — into full production on June 1, 2026, claiming 1.8x faster task completion vs. x86 and "a market that never existed before." This validates the theme but also activates the thesis's own invalidation trigger (Vera-as-standalone displacing the host CPU and compressing x86). A new server OEM/ODM layer (L5) emerged as the cleanest fresh exposure surface for the standalone-CPU buildout. v2 note: hyperscalers consolidated into a new top-of-stack L0; L7 renamed Neocloud / Infra hosts.
